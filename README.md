### Instant-Meshes-Core

This is a headless C++ library refactored from the [instant-meshes](https://github.com/wjakob/instant-meshes) repo.
It provides core functionalities described in [1] without any graphical user interface (GUI).


[1]: Jakob, Wenzel, et al. "Instant field-aligned meshes." ACM transactions on graphics (TOG) 34.6 (2015): 1-15.
